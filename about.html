About Me

My background is in Cloud Engineering and deploying Cloud Infrastructure. I have 4 years of experience in Cloud technologies working on both Azure and AWS cloud platforms.

In my previous role, I worked on a ServiceNow integration project where I automated workflows and streamlined IT service management processes. This project involved integrating ServiceNow with various cloud services to enhance operational efficiency.

I have extensive experience with Azure, where I utilized Infrastructure as Code (IaC) tools like Terraform to build and deploy scalable cloud solutions. One of my key projects involved deploying Azure OpenAI services using Terraform, ensuring consistent and repeatable infrastructure provisioning. This allowed for seamless integration of AI capabilities into enterprise applications while maintaining robust security and compliance standards.

In my free time, I enjoy learning new programming languages and frameworks, as well as contributing to open-source projects. I also love to travel and explore new cultures.
